# <p align="center">Scrabble
<p align="center">Scrabble game in C#, using <a href="https://en.wikipedia.org/wiki/Windows_Presentation_Foundation">WPF</a>.

## Description
* Standard 100-pieces version of Scrabble
* No "Challenge" option
* No "The Fifty Point Bonus"
